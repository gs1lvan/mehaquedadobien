📱 Web Share API

Esta funcionalidad usa la Web Share API del navegador para compartir archivos XML de recetas.

Estado de soporte:

🎯 Características Implementadas

  • Botón "🔗 Compartir" en el header (junto a "Exportar todas las recetas")
  • Botón "🔗 Compartir" en la vista de detalle de cada receta
  • Compartir a través de apps nativas (Drive, Gmail, WhatsApp, etc.)
  • Fallback automático a descarga si el navegador no soporta compartir
  • Solo visible en navegadores compatibles
  • Funciona en móvil y desktop (Chrome, Edge, Safari)

🧪 Prueba la Funcionalidad

📋 Cómo Usar

  1. En la lista de recetas:
    • Click en "🔗 Compartir" (junto a "Exportar todas las recetas")
    • Comparte todas las recetas o las filtradas
  2. En el detalle de una receta:
    • Click en "🔗 Compartir" (junto a los botones de exportar)
    • Comparte solo esa receta
  3. Elige dónde compartir:
    • 📧 Gmail / Email
    • 💾 Google Drive
    • 💬 WhatsApp
    • 📱 Otras apps instaladas

🌐 Compatibilidad de Navegadores

Navegador Desktop Móvil
Chrome ✅ v89+ ✅ v61+
Edge ✅ v89+ ✅
Safari ✅ v12.1+ ✅ v12.2+
Firefox ❌ ✅ v71+

* En navegadores no compatibles, se descarga el archivo automáticamente

⚠️ Nota Importante

La funcionalidad de compartir requiere HTTPS o localhost para funcionar por razones de seguridad.

Si abres el archivo directamente con file://, solo verás el fallback de descarga.
